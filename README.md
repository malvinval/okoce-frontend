## Installation Guides

1. **Clone this repository to your local machine**

    Using SSH:

    `git clone git@github.com:malvinval/okoce-frontend.git`

    or using HTTPS:

    `https://github.com/malvinval/okoce-frontend.git`

2. **Go to the project directory**

    `cd <directory_name>`

3. **Install dependencies**

    `npm install`
